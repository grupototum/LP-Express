# Project Memory

## Core
- **Tech Stack**: React + Vite + Tailwind + Framer Motion (no Next.js, no GSAP on this project).
- **Brand**: Atelier Rosso — independent creative studio. Premium dark, cinematic.
- **Tokens**: Rosso `#da291c` (primary/accent), Canvas `#0a0a0a` (background), white text.
- **Typography**: Bodoni Moda (display, uppercase, tight tracking, italic for accents) + Inter / Neue Haas Grotesk (body). Eyebrows in `text-[11px] tracking-[0.3em] uppercase`.
- **Layout**: Max `1440px` container, generous negative space, full-bleed photography, grayscale → color hover on imagery.
- **Routing**: `/` → AtelierRosso. `/express` → `<Navigate to="/" replace />` (canonical consolidation).
- **Sections (in order)**: Nav · Hero · Reel · Selected Work · Capabilities · Clients · Contact · Footer.
